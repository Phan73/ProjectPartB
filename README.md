Website Plan
Website Title: Toys and Fun

1. Topic Overview
This project focuses on developing an e-commerce website for a toy store. The site will feature a wide range of toys suitable for children of various age groups, including:

Educational toys

Puzzles and board games

Outdoor toys

Action figures

Dolls and plush toys

The goal is to create a user-friendly and visually appealing website that encourages browsing, supports secure purchases, and builds trust with parents and caregivers.

2. Proposed Website URL
toysandfun.co.nz
This domain has been selected because it is short, easy to remember, clearly reflects the brand identity, and is currently available.

4. Page Designs
Home Page / About Us Page
These pages will share a consistent layout:

Header with logo and navigation bar

Hero image or promotional banner

Welcome message or brand introduction

Brief overview of the store (About Us) or highlighted toy categories (Home)

Footer with links and contact info

Category Pages
Each category page will include:

Navigation bar and logo

Greeting message or short intro text

Ad banner or promotional deal section

Product listing grid with images, names, prices, and “Add to Cart” buttons

Contact Us Page
This page will feature:

Navigation bar and logo

Sectioned content layout:

Business contact information (address, phone, email)

Category links for general inquiries, shipping, returns, etc.

Contact Form including:

Name

Email

Subject

Message box

Submit button
